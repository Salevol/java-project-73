### Project description
Task Manager is a simple web-app to track tasks.
<a href="https://java-project-73-production-9178.up.railway.app/">Deployed app</a>.

Project stack:
- Spring Boot
- Spring Security
- Hibernate
- Swagger
- Rollbar

To run locally:
```sh
make run
```
or 
```shell
docker compose up
```
